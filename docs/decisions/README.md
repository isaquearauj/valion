# Architecture Decision Records

ADRs registram decisões duráveis que afetam várias mudanças futuras. Não substituem specs locais:
uma spec orienta uma entrega; um ADR explica por que o projeto adotou uma direção arquitetural.

Use o formato `NNNN-titulo-em-kebab-case.md` com contexto, decisão, consequências e alternativas.
Não crie ADR para preferência local ou detalhe facilmente reversível.
